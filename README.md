# Machine Learning Projects

This repository serves as an index for all my ML-related projects.

- [🤖⚙️AutoML Tool](https://github.com/Mohab-DataAnalyst/autoML)
  - End-to-end AutoML-style web application for uploading tabular datasets, reviewing data quality recommendations, training multiple machine learning models, comparing results, and exporting the best trained model

- [📝🎓Exam Score Prediction](https://github.com/Mohab-DataAnalyst/exam-score-prediction)
  - Uses regression modelling to predict a student's exam score from study habits, lifestyle factors, attendance, and background information, and deploys with FastAPI and Streamlit

- [👥🚪Customer Churn Prediction](https://github.com/Mohab-DataAnalyst/customer-churn-prediction)
  - Uses classification modelling to predict whether a customer is likely to churn based on his profile, contract, service, tenure, and billing information, and deploys with FastAPI and Streamlit

- [🎬🍿Movie Recommendation System](https://github.com/Mohab-DataAnalyst/movie-recommender-system)
  - Builds a movie recommendation system using ratings and genre data. It includes collaborative filtering, content-based recommendations, matrix factorization, and deployment using FastAPI and Streamlit
